## Major Feature Release

We're excited to announce our biggest update yet! This release brings powerful new features that will transform how you manage your business, from quote creation to enhanced user preferences. We've also made significant improvements to our infrastructure and user experience.

- Complete quote management system with conversion capabilities
- Enhanced user preferences and notification controls
- Improved invoice limits and usage tracking

## New Features

### Quote Management System

Create professional quotes that convert seamlessly into invoices. Our new quote system provides the same intuitive interface as invoice creation, with specialized features for quote management.

- Quote creation with professional templates
- One-click quote-to-invoice conversion
- Automated quote expiration and cleanup
- Custom email templates for quote delivery
- Separate dashboard for active and draft quotes

### Enhanced User Experience

We've significantly improved how you interact with Billo through better preferences, notifications, and usage tracking.

- Default invoice terms and template selection
- Comprehensive notification preferences
- Monthly invoice usage tracking with plan limits
- Enhanced branding controls for paid users
